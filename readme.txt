== Description ==

Dieses Plugin erstellt ein Sidebar-Widget mit welchem BezahlCodes generiert werden koennen.
Weitere Informationen zum BezahlCode unter www.bezahlcode.de

== Installation ==

1. Plugin in das Plugin-Verzeichnis ("/wp-content/plugins/") kopieren
2. Modul aktivieren im Wordpress Admin-Bereich
3. im Admin-Bereich unter Design -> Widget findet man nun das Widget "BezahlCode-Widget"
